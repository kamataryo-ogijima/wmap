# 男木島地図アプリ

[![Build Status](https://travis-ci.org/ogijima-library/wmap.svg?branch=master)](https://travis-ci.org/ogijima-library/wmap)

https://ogijima-library.github.io/wmap/


```
$ git clone git@github.com:ogijima-library/wmap.git
$ cd wmap
$ npm install
```

## Builds the app

```
$ npm run build
```

## Start the server

```
$ npm start
```
